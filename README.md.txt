# LensParse.ai
### Intelligent Multimodal OCR & Document Intelligence Engine

LensParse is a high-performance document parsing utility powered by **Gemini 2.5 Flash**. It converts heterogeneous document assets (Invoices, Resumes, IDs) into structured, actionable JSON data with high spatial accuracy.

## Features
- **Zero-Shot Extraction**: No pre-defined templates required.
- **Multimodal Intelligence**: Leverages Gemini 2.5 Flash for complex layout analysis.
- **Professional UI**: Built with a modern Glassmorphism aesthetic using Streamlit.
- **Export Ready**: One-click JSON downloads for integration into downstream workflows.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Create a `.env` file and add your `GOOGLE_API_KEY`.
4. Run the app: `streamlit run app.py`.
